JavaScript-htpasswd
===================

JavaScript-htpasswd is a small experiment (and somewhat useful utility) to
generate `.htpasswd` entries in your browser using client-side JavaScript.

Features
--------
- Client-side password hash generation using CryptoJS
- Supported hashes:
    - Plain text
    - Unsalted SHA1 (SHA)
    - Salted SHA1 (SSHA)
